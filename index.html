I'm Kristina Jikia, a Frontend Developer and Product Manager originally from Tbilisi, Georgia, currently based in France. I work across the full product lifecycle — from writing React & TypeScript code to crafting PRDs, Risk Registers and conducting structured User Research with 40+ participants.

Key Product Decisions
PWA over Native App — Saved native development costs while delivering offline mode, browser-based push notifications and home screen installability without App Store friction.
MoSCoW Prioritization — GitHub/Figma submission, PWA install and role-based auth as Must-Have; AI-driven analytics and native video chat deferred to Phase 2.
Risk R.01 — iOS PWA — Identified Push Notification failure on iOS < 16.4. Designed Email/SMS Fallback via Twilio as contingency plan, preventing deadline misses for iOS users.
North Star Metric
"Active Weekly Users with 100% Submission Rate"

This single metric unifies all three stakeholder goals: Admin has the system configured and running, Lecturer has uploaded assignments on time, Student used the PWA and submitted before the deadline. If this number grows, everything is working.
